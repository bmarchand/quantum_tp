

Bonjour à tous,

Le TP de programmation quantique de ce vendredi 8/01 (dans le cadre du cours de technologies quantiques) se fera en Python3, au sein d'un notebook Jupyter, en utilisant la librairie de programmation quantique myqlm (développée par l'entreprise Atos: https://myqlm.github.io/index.html).

Vous utiliserez vos machines personnelles, sur lesquelles vous devez donc installer python3,  jupyter-notebook et myqlm (y compris les "jupyter magics" propres à myqlm).

Le test pour savoir si vous avez installé tous les prérequis nécessaires au TP est de réussir à faire tourner le "minimal_notebook" ci-joint (i.e vous devez réussir à ouvrir le notebook et faire tourner sans erreurs toutes ses cellules). L'idéal serait que vous réussissiez à le faire tourner d'ici vendredi. Je suis à votre disposition par mail ou par teams pour répondre à vos questions concernant l'installation.

Je vous également joint une version html pour que vous voyiez à quoi ressemble un notebook ayant été exécuté avec succès.

Si vous ne savez pas ce qu'est un jupyter-notebook: https://github.com/jupyter/jupyter/wiki/A-gallery-of-interesting-Jupyter-Notebooks .

Ci-dessous quelques éléments d'aides à l'installation spécifiques à votre plateforme:

LINUX/MAC:

1. Ouvrez un terminal.

2. Vérifiez votre version de python en tapant "python3 --version", ainsi que votre version de pip3 ou pip (l'installateur de package python): "pip3 --version". Vérifiez la concordance entre ces deux versions et avec les prérequis de myqlm: https://myqlm.github.io/myqlm_specific/install.html#prerequisites

(il vous faut python3.6 ou python3.8 sur linux, 3.6 3.7 ou 3.8 sur mac)

3. Suivez les instructions de https://myqlm.github.io/myqlm_specific/install.html#prerequisites

4. exécutez: "jupyter-notebook --no-browser --port 8080" et ouvrez le lien qui s'affiche sur votre terminal. Naviguez jusqu'au minimal_notebook, ouvrez-le, et essayez d'éxécuter ses cellules. Vous pouvez faire Kernel --> restart and run all pour tout lancer d'un coup. Si il n'y a pas d'erreurs vous avez terminé.

WINDOWS:

Le plus simple, que j'ai pu tester sur une machine windows avec succès, serait de:

1. Installez anaconda individual edition: https://www.anaconda.com/products/individual

(c'est un peu volumineux: ~500Mb, mais tout se fait très facilement ensuite)

2. lancez "anaconda navigator".

3. cliquez-sur "CMD command prompt".

4. éxécutez "pip install myqlm".

5. comme expliqué sur https://myqlm.github.io/myqlm_specific/install.html#prerequisites, installez ImageMagick. Puis éxécutez "python -m qat.magics.install"

6. retournez sur "anaconda navigator" et cliquez sur "launch jupyter notebook". Naviguez sur le minimal_notebook et tout devrait marcher.
